# Импорт таблиц в CineVault

CineVault принимает файлы `.csv`, `.tsv` и `.xlsx`. Для XLSX приложение
должно быть запущено через `launch.py`, поскольку локальный Python-сервер
извлекает данные из первого листа файла.

## Минимальный формат

Обязателен только один столбец:

```text
Название
```

Также распознаются следующие заголовки:

| Поле | Допустимые заголовки |
|---|---|
| Название | `Название`, `Фильм`, `Title`, `Movie` |
| Оригинальное название | `Оригинальное название`, `Original title` |
| Категория | `Категория`, `Category`, `Подборка` |
| Франшиза | `Франшиза`, `Franchise`, `Серия` |
| Год | `Год`, `Year`, `Release year` |
| Продолжительность | `Длительность`, `Продолжительность`, `Duration`, `Минуты` |
| Страна | `Страна`, `Country` |
| Постер | `Обложка`, `Постер`, `Cover`, `Poster`, `Cover URL` |
| Статус | `Просмотрено`, `Просмотрен`, `Watched`, `Status` |
| Дата просмотра | `Дата просмотра`, `Watched at`, `WatchedDate` |

## Вложенные категории

Путь категории можно записать через `>`, `/` или `\`:

```text
Фантастика > Космос
Фэнтези / Средиземье
```

Недостающие категории и подкатегории будут созданы автоматически.

## Оценки

Каждому зрителю соответствует отдельный столбец:

```text
Оценка Антон
Оценка Иван
Rating: Alex
```

Поддерживаются значения от 1 до 10 с шагом 0,5. Десятичный разделитель
может быть точкой или запятой.

## Пример

```csv
Название;Категория;Франшиза;Год;Просмотрено;Оценка Антон
Братство кольца;Фэнтези > Средиземье;Властелин колец;2001;да;9,5
Две крепости;Фэнтези > Средиземье;Властелин колец;2002;;9
```

Импорт объединяет таблицу с текущей библиотекой. Совпадающие фильмы не
создаются повторно.
