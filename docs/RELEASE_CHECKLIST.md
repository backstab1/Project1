## Известные ограничения

- Данные IndexedDB принадлежат конкретному профилю браузера.
- Для переноса между компьютерами используется JSON-резервная копия.
- XLSX импортируется с первого листа.
- Формулы XLSX читаются по сохранённому значению ячейки.
- Автоматическое получение метаданных требует будущего онлайн-модуля.
