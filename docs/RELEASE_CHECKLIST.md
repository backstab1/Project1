# CineVault 0.9.0-beta — чек-лист приёмки

## Автоматически проверено

- [x] JavaScript-модули проходят синтаксическую проверку.
- [x] Python-лаунчер проходит `py_compile`.
- [x] Доменная бизнес-логика покрыта Node test runner.
- [x] Парсер XLSX покрыт Python `unittest`.
- [x] Полный сценарий проходит от формирования библиотеки до восстановления
      резервной копии.
- [x] Portable-папка и ZIP создаются `build-portable.ps1`.
- [x] Невалидный XLSX отклоняется локальным API с HTTP 400.

## Известные ограничения

- Данные IndexedDB принадлежат конкретному профилю браузера.
- Для переноса между компьютерами используется JSON-резервная копия.
- XLSX импортируется с первого листа.
- Формулы XLSX читаются по сохранённому значению ячейки.
- Автоматическое получение метаданных требует будущего онлайн-модуля.
