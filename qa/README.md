# Visual QA

Каталог `qa/` содержит воспроизводимую визуальную проверку CineVault и не
включается в пользовательскую Windows-сборку.

`seed.html` создаёт отдельную тестовую библиотеку с вложенными категориями,
длинными русскими названиями, франшизой, просмотренными фильмами и оценками.
`capture-visuals.ps1` запускает отдельный профиль headless Edge, дожидается
завершения IndexedDB-операций и снимает ключевые экраны при 1366×768.

Запуск:

```powershell
powershell -ExecutionPolicy Bypass -File .\qa\capture-visuals.ps1
```

Скрипт также завершится с ошибкой, если какой-либо экран создаёт
горизонтальное переполнение. Снимки и временный профиль сохраняются в
игнорируемом Git каталоге `release/`.
